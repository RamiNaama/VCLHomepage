TESTCUBE

TestCUBE is a hardware and software solution from Softing company for simulating ECU diagnostics with real bus communication. It can be used to simulate several ECUs or even an entire vehicle at the same time. TestCUBE is a valuable tool for preparing tests in engineering, testing, and manufacturing, as well as for tester or production system release tests and tester regression tests.
